**1. `getRevenueConcentration(topN = 5): Promise<RevenueConcentration[]>` (DERIVE-01, D-02)**

Interface:
```typescript
export interface RevenueConcentration {
  fee_category: string;
  total_service_charges: number;  // dollars (scaled * 1000 in SQL per D-09)
  share_pct: number;              // 0-100
  cumulative_pct: number;         // monotonically increasing
  institution_count: number;
}
```

SQL approach: CTE `category_totals` joins `extracted_fees` (fee_category) with `institution_financials` (service_charge_income * 1000) via crawl_target_id. Filter: `ef.fee_category IS NOT NULL AND ef.review_status != 'rejected' AND ifin.service_charge_income IS NOT NULL`. Use latest report_date per institution (subquery). Compute share_pct as `total_sc * 100.0 / grand_total`. ORDER BY total_sc DESC LIMIT $1.

Compute cumulative_pct in TypeScript via `.reduce()` after SQL returns (per research pitfall #3).

Use `sql.unsafe(query, [topN])` for the parameterized LIMIT (per anti-pattern guidance: do NOT use sql template tag for positional params).

**2. `getFeeDependencyRatio(opts?: { charter?: string; tier?: string }): Promise<FeeDependencyRow[]>` (DERIVE-02, D-04)**

Interface:
```typescript
export interface FeeDependencyRow {
  charter_type: string;
  asset_size_tier: string;
  median_ratio: number;           // SC / total_revenue median
  p25_ratio: number;
  p75_ratio: number;
  institution_count: number;
  total_sc_income: number;        // dollars (per D-09, * 1000)
  overdraft_revenue: number | null; // dollars (per D-08, may be null)
  other_sc_income: number | null;   // total_sc - overdraft (per D-08)
  overdraft_share: number | null;   // overdraft/total_sc pct (per D-08)
}
```

SQL: Query institution_financials JOIN crawl_targets. Group by charter_type + asset_size_tier (from crawl_targets). For each group, collect fee_income_ratio values and compute percentiles in TypeScript (sorted array approach, matching health.ts pattern). Apply `* 1000` to service_charge_income and overdraft_revenue. Compute overdraft_share = overdraft_revenue / service_charge_income * 100 when both non-null. Use optional WHERE clauses for charter and tier filters.

**3. `getRevenuePerInstitution(): Promise<RevenuePerInstitutionRow[]>` (DERIVE-03)**

Interface:
```typescript
export interface RevenuePerInstitutionRow {

  median_sc_income: number;       // dollars

}
```

SQL: `SELECT ct.charter_type, ct.asset_size_tier, AVG(ifin.service_charge_income * 1000) as avg_sc, COUNT(DISTINCT ifin.crawl_target_id) as inst_count FROM institution_financials ifin JOIN crawl_targets ct ON ct.id = ifin.crawl_target_id WHERE ifin.service_charge_income IS NOT NULL AND ifin.report_date = (latest per institution subquery) GROUP BY ct.charter_type, ct.asset_size_tier`. Compute median in TypeScript from per-institution values.
