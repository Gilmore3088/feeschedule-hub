Actions:
- `createIntelligence(formData: FormData): Promise<{ success: boolean; error?: string }>` — Extract source_name, source_date, category, tags (comma-split + trim), content_text from formData. Validate: source_name and content_text required, content_text min 50 chars. Call `insertIntelligence(...)` with `created_by` from `getCurrentUser()?.username`. Return `{ success: true }` or `{ success: false, error }`.

- `fetchUrlContent(url: string): Promise<{ content: string; error?: string }>` per D-02 — Validate URL starts with http(s). Use `fetch(url, { signal: AbortSignal.timeout(10000) })` to get HTML. Strip tags with `sanitize-html(html, { allowedTags: [], allowedAttributes: {} })` to extract plain text. Trim to first 50,000 chars. Return `{ content }` or `{ error }`. Wrap in try/catch for network errors.

- `listIntelligenceAction(page?: number): Promise<{ items: ExternalIntelligence[]; total: number }>` — Calls `listIntelligence(50, (page ?? 0) * 50)`.

- `deleteIntelligenceAction(id: number): Promise<{ success: boolean }>` — Calls `deleteIntelligence(id)`. Revalidate path.

IntelligencePanel (server component)

**IntelligenceForm** (client component, defined in same file with "use client" boundary):
- Fields: source_name (text input, required), source_date (date input, required, default today), category (select: research/survey/regulation/news/analysis), tags (text input, comma-separated), content mode toggle (paste text / fetch URL)

- On success: clear form, show success message, router.refresh()
- On error: show error message in red text
- Styling: `.admin-card` wrapper, Geist font conventions, form inputs with `rounded border border-gray-200 px-3 py-2 text-sm` pattern

- Each row: `.admin-card` or table row with hover
- Delete button: small red text button, calls deleteIntelligenceAction
- Tags rendered as small gray pills
- Total count shown above list: "{N} documents"
