## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-40-04 | Information Disclosure | HamiltonLeftRail showing other users' data | mitigate | All DB queries in server layout filter by user_id from authenticated session; client component only receives filtered results as props |
| T-40-05 | Elevation of Privilege | Admin nav link visible to non-admins | mitigate | HamiltonTopNav receives isAdmin prop; Admin link conditionally rendered only when isAdmin=true |
| T-40-06 | Spoofing | Upgrade gate bypass via direct component import | accept | Gate renders in server layout before children; client cannot bypass server-side auth check |
| T-40-07 | Information Disclosure | Institution context leaking between users | mitigate | institutionContext derived from authenticated user's own profile fields in server layout; no cross-user data exposure |
</threat_model>
