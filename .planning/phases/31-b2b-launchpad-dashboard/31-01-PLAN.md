  <action>
In the same `src/app/pro/dashboard.tsx` file (continuing from Task 1), implement the peer snapshot sidebar in the `lg:col-span-4` right column.

**Peer snapshot data fetching (at top of ProDashboard function):**
1. Get district number: `const district = user.state_code ? STATE_TO_DISTRICT[user.state_code] : null`
2. Build peer filters from user profile:
   ```
   const peerFilters: { charter_type?: string; asset_tiers?: string[]; fed_districts?: number[] } = {};
   if (user.institution_type) peerFilters.charter_type = user.institution_type === 'credit_union' ? 'credit_union' : 'bank';
   if (user.asset_tier) peerFilters.asset_tiers = [user.asset_tier];
   if (district) peerFilters.fed_districts = [district];
   ```
3. Fetch: `const [nationalIndex, peerIndex] = await Promise.all([getNationalIndexCached(), getPeerIndex(peerFilters)])`
4. Get spotlight categories: `const spotlightCats = getSpotlightCategories().slice(0, 3)` (top 3 only)
5. Build comparison data:
   ```
   const peerComparison = spotlightCats.map(cat => {
     const national = nationalIndex.find(e => e.fee_category === cat);
     const peer = peerIndex.find(e => e.fee_category === cat);
     if (!national?.median_amount || !peer?.median_amount) return null;
     const delta = ((peer.median_amount - national.median_amount) / national.median_amount) * 100;
     return { category: cat, peerMedian: peer.median_amount, nationalMedian: national.median_amount, delta };
   }).filter(Boolean);
   ```

**Peer snapshot panel UI (right sidebar):**
