       def _is_blacklisted(url: str) -> bool:

           hostname = urlparse(url).hostname or ""

       ```python
       def probe_pdf_urls(self, base_url: str, rate_limiter=None) -> list[str]:
           """Probe common PDF paths on a website to find direct fee schedule PDFs.

           Returns list of discovered PDF URLs (may be empty).
           Uses HEAD requests first to avoid downloading full PDFs.
           """
           found = []
           parsed = urlparse(base_url)
           origin = f"{parsed.scheme}://{parsed.netloc}"

           for path in PDF_DIRECT_PROBE_PATHS:
               probe_url = origin + path
               if _is_blacklisted(probe_url):
                   continue
               try:
                   if rate_limiter:
                       rate_limiter.wait(probe_url)
                   resp = self.session.head(
                       probe_url, timeout=10, allow_redirects=True,
                   )
                   content_type = resp.headers.get("Content-Type", "").lower()
                   if resp.status_code == 200 and "application/pdf" in content_type:
                       found.append(probe_url)
                       break  # first PDF found is enough
               except Exception:
                   continue
           return found
       ```

|----------|-------------|

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-59-01 | Tampering | SSRF via probed PDF URLs | mitigate | Existing `_is_safe_url()` in both `download.py` and `playwright_fetcher.py` blocks RFC 1918, link-local, and cloud metadata IPs -- all discovered URLs pass through this check before fetch |
| T-59-02 | Tampering | Malicious PDF exploiting pdfplumber | accept | pdfplumber does not execute code; extracted text is passed to LLM as string; low risk for text extraction library |
| T-59-03 | Spoofing | Stealth browser impersonation | accept | Stealth is used to access publicly published fee schedules that institutions make available to consumers; ethical use for data that is intended to be public |
| T-59-04 | Information Disclosure | Probed URLs reveal crawler behavior | accept | HEAD-first probing is standard web client behavior; rate limiting prevents detection as scanner |
| T-59-05 | Denial of Service | Unbounded PDF probe requests | mitigate | PDF probe uses `break` on first successful result; rate_limiter controls per-domain request spacing; 14-path list is bounded |
