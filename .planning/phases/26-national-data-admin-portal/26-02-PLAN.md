## New Data Sources for 26-02

### CFPB Complaints (Phase 25 data layer)
From `src/lib/crawler-db/complaints.ts`:
```typescript
getDistrictComplaintSummary(district: number): Promise<DistrictComplaintSummary>
  → {fed_district, total_complaints, fee_related_complaints, institution_count, top_products}

getInstitutionComplaintProfile(crawl_target_id): Promise<InstitutionComplaintProfile>
  → {total_complaints, by_product, by_issue, fee_related_pct}

getNationalComplaintSummary(): Promise<{
  total_complaints: number,
  fee_related_pct: number,
  average_per_institution: number
}>
```

**Usage in 26-02:**
- Economic panel: Show district-level complaint counts alongside FRED indicators
- Add "CFPB Complaints" card showing national total + fee-related %
- Comparison metric: "District X has Y complaints vs. national average of Z"

### Derived Analytics (Phase 25 data layer)
From `src/lib/crawler-db/derived-analytics.ts`:
```typescript
getRevenueConcentration(): Promise<RevenueConcentration>
  → {
    dollar_volume: [{fee_category, value, pct_of_total}],
    institution_prevalence: [{fee_category, value, pct_of_total}],
    summary: {top_n: 5, dollar_volume_pct: X, prevalence_pct: Y, total_fee_dollars, total_institutions}
  }

getFeeDependencyTrend(): Promise<FeeDependencyTrend>
  → {
    trend: [{report_date, avg_fee_income_ratio, institution_count}],
    signals: {qoq_change_pct, yoy_change_pct}
  }

getRevenuePerInstitutionTrend(): Promise<RevenuePerInstitutionTrend>
  → {
    current: {by_tier: [], by_charter: []},
    trend: [{report_date, avg_sc_income, institution_count}],
    signals: {qoq_change_pct, yoy_change_pct}
  }
```

**Usage in 26-02:**
- Overview panel: Add callout card "Revenue Concentration: Top 5 fee categories = X% of national service charges"
- Overview panel: Add callout "Fee Dependency: X% of revenue from fees (Y% QoQ, Z% YoY)"
- Call Reports panel: Add section showing "Top 5 Fee Categories by Revenue Volume" with % breakdown chart
