## Data Layer (Phase 23-25)

### Call Reports (Phase 23)
- `getRevenueTrend(quarters: number)` — Returns {latest, trend, yoy_change_pct}
- `getRevenueByCharter()` — Bank vs CU split with institution counts
- `getTopInstitutionsByRevenue(limit: number)` — Name, total assets, charter, service_charge_income (in dollars, not thousands)

### FRED Indicators (Phase 23)
- `getNationalEconomicSummary()` — Returns {fed_funds_rate, unemployment_rate, cpi_yoy_pct, consumer_sentiment} as RichIndicator objects
- RichIndicator shape: {current: number, asOf: string, trend: "rising"|"falling"|"stable", history: {value: number}[]}

### Industry Health (Phase 24)
- `getIndustryHealthMetrics()` — Returns {roa, roe, efficiency_ratio} as RichIndicator objects (same shape as FRED)

### Beige Book (Phase 24)
- `getBeigeBookHeadlines()` — Returns Map<districtNum, {text, release_date}>
- `getDistrictBeigeBookSummaries()` — Returns [{district_number, district_name, summary, themes, release_date}]

### Derived Analytics (Phase 25, for 26-02 planning)
- `getRevenueConcentration()` — {dollar_volume: [], institution_prevalence: [], summary: {top_n, dollar_volume_pct, ...}}
- `getFeeDependencyTrend()` — {trend: [{report_date, avg_fee_income_ratio, institution_count}], signals: {qoq, yoy}}
- `getRevenuePerInstitutionTrend()` — {current: {by_tier, by_charter}, trend: [], signals}

### CFPB Complaints (Phase 25, for 26-02)
- `getDistrictComplaintSummary(district)` — Returns total_complaints, fee_related_complaints, institution_count, top_products
- `getInstitutionComplaintProfile(crawl_target_id)` — Returns total_complaints, by_product, by_issue, fee_related_pct

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Admin → Database | Admin-only queries return aggregated/non-PII financial data; queries are read-only |
| Admin → Public Data | Call Reports, FRED, Beige Book are all public data sources; no credential leakage |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-26-01 | Information Disclosure | Overview panel displays aggregated revenue | Accept | Data is public (Call Reports published quarterly by FDIC/NCUA); no PII exposed |
| T-26-02 | Availability | Panel queries timeout/fail | Mitigate | All queries wrapped in .catch(() => null); panel gracefully shows "No data available" instead of crashing |
| T-26-03 | Integrity | Stale economic data displayed as current | Mitigate | FreshnessBadge component shows data age; red badge if > 30d stale; tooltip indicates when data was last updated |
| T-26-04 | Denial of Service | Admin spam-refreshing `/admin/national` overwhelms DB | Accept | Read-only queries; admin users are authenticated; no public endpoint affected |
</threat_model>
