    **`run_single_state(conn, wave_run_id: int, state_code: str) -> dict | None`**:
    - update_wave_state(conn, wave_run_id, state_code, status="running")
    - Call run_state_agent(state_code) — this is synchronous, returns summary dict
    - On success: update_wave_state(conn, wave_run_id, state_code, status="complete", agent_run_id=result["run_id"])
    - On HARD_FAILURES: re-raise (campaign should stop)
    - On any other Exception: update_wave_state(conn, wave_run_id, state_code, status="failed", error=str(e)[:500]), log.error, return None
    - Log timing: record wall-clock seconds per state

    **`run_wave(conn, states: list[str], wave_size: int | None = None, campaign_id: str | None = None) -> int`**:
    - Per D-01: Execute states SEQUENTIALLY (not parallel) for maximum reliability on unattended multi-hour runs
    - ensure_tables(conn) at start
    - wave = create_wave_run(conn, states, wave_size or len(states), campaign_id)
    - update_wave_run(conn, wave.id, status="running")
    - For each state in states:
      - result = run_single_state(conn, wave.id, state)
      - If result: completed_states += 1, else: failed_states += 1
      - update_wave_run(conn, wave.id, completed_states=completed, failed_states=failed)
    - Per D-04: On HARD_FAILURES, update_wave_run status="failed", re-raise
    - On completion: update_wave_run status="complete", completed_at=now
    - Log wave summary: "Wave #{wave.id}: {completed}/{total} states complete, {failed} failed"
    - Return wave.id

    **`resume_wave(conn, wave_run_id: int) -> int`**:
    - Per WAVE-04: Get incomplete states via get_incomplete_states(conn, wave_run_id)
    - If none: log "All states complete", return wave_run_id
    - update_wave_run(conn, wave_run_id, status="running")
    - Run each incomplete state via run_single_state()
    - Same error handling as run_wave
    - Return wave_run_id

    - Collect wave_ids
    - Per D-04: If a wave raises a hard failure, stop campaign, log which wave failed
