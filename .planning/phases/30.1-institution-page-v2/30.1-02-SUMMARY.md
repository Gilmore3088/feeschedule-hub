# Phase 30.1 Plan 02: Page Restructure + Table Indicators Summary

**One-liner:** Institution page restructured to V2 visual hierarchy with rating-first layout, directional table arrows, mid-page CTA, and compare hooks linking to filtered institution search.

## What Was Built

### Task 1: MidPageCTA + CompareSection components
Two new server components created for the V2 layout:

**`mid-page-cta.tsx`** — D-15 locked copy card that appears after the visual comparison section for non-pro users. Links to `/for-institutions`. No UpgradeGate per D-17.

**`compare-section.tsx`** — Renders 2-3 pill links to the existing institution search page with pre-applied filters:
- "Compare to nearby institutions" → `/institutions?state={stateCode}`
- "Compare to {charterLabel} in {stateName}" → `/institutions?state={stateCode}&charter={charterType}`
- "Browse all institutions" → `/institutions` (always shown)

Both components are server-only (no `"use client"`). State code and charter type are URL-encoded before use in hrefs (T-30.1-05).

### Task 2: Page restructure + table indicator enhancement

**V2 section ordering** implemented in `page.tsx`:

1. Breadcrumb (unchanged)
2. Institution name + subtitle (unchanged, left-aligned)
3. Source links (unchanged)
4. **FeeSummaryCard** — most prominent, centered, green/yellow/red rating label + bullets
5. **InterpretationBlock** — 2-3 plain English sentences from `generateInterpretation()`
6. **FeeComparisonBars** — hidden when zero category matches (D-06)
7. **FeeCountCard** — institution count vs typical bank/CU ranges
8. **ProsConsBlock** — hidden when no strengths/watch data (D-09)
9. **MidPageCTA** — only for non-pro users (D-15, D-17)
10. Info cards (charter, tier, assets, fee count)
11. Financial Snapshot (pro only)
12. Market Context (pro only)
13. Competitive Scorecard (unchanged)
14. Financial Context / Call Reports (pro only)
15. Fee Distribution (unchanged)
16. **Fee Schedule table** — enhanced with directional arrows
17. **CompareSection** — filtered search links
18. Intelligence & Reports / footer CTA

**Fee table enhancements (D-07):**
- `↑` in `text-red-600` when delta > 0.5% (above median)
- `↓` in `text-emerald-600` when delta < -0.5% (below median)
- `=` in `text-[#A09788]` when |delta| ≤ 0.5% (at median)
- Empty cell (no character) when no category match
- National Median and vs. Median columns hidden entirely when zero fees have a match

**Footer CTA (D-16):** Updated copy to "Financial professionals use Bank Fee Index to benchmark pricing, analyze revenue, and identify opportunities."

**UpgradeGate:** Import removed, all 0 references in page.tsx.
