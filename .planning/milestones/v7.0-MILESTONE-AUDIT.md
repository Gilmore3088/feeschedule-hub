## E2E Flows

### Flow 1: Report Generation with Thesis
1. `assembleNationalQuarterly()` builds payload
2. `buildThesisSummary()` condenses to ~5KB
3. `generateGlobalThesis()` calls Claude (90s timeout)
4. `thesisContext` injected into all 6 section contexts
5. Sections generated in parallel via `Promise.allSettled()`
6. `runEditorReview()` validates thesis alignment, revenue ordering, implications
7. Report rendered
- **Status:** Complete

### Flow 2: Unified Chat
1. User hits `/api/research/hamilton`
2. `getCurrentUser()` resolves role
3. `getHamilton(role)` builds agent with role-appropriate tools/model/prompt
4. Daily cost circuit breaker checked
5. Streaming response via AI SDK
- **Status:** Complete

### Flow 3: Regulatory Risk Query
1. Hamilton receives compliance question
2. Calls `queryRegulatoryRisk` tool
3. Cross-references fee outliers + CFPB complaints + Fed speeches
4. Returns risk score (0-100) with affected institutions
- **Status:** Complete

**Flows: 3/3 verified**
