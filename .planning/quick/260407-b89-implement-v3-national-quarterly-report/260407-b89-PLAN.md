```sql
SELECT value, observation_date
FROM fed_economic_indicators
WHERE series_id = 'CPIAUCSL'

LIMIT 13
```

3. If fewer than 13 rows, fall back to null for cpi_yoy_pct.

```ts
  provisional_maturity_count: number;
```

**Compute derived after building categories array:**
1. For each category with non-null median and p25/p75, compute `spread_pct = ((p75 - p25) / median) * 100`. Filter out categories with median near zero.
2. Sort by spread_pct ascending for tightest (top 5), descending for widest (top 5).
3. `commoditized_count` = count where spread_pct < 30.
4. `avg_iqr_spread_pct` = average of all spread_pct values.
5. Bank vs CU: filter categories with both bank_count > 0 and cu_count > 0 and non-null medians. Compare medians. Sort by absolute diff_pct for premiums.
6. Revenue: if payload.revenue exists, compute per-institution and share percentages.
7. Maturity: count strong and provisional.

Add `derived` to the return object.
